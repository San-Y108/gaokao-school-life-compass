# AGENTS

## 项目目标
开发一个面向高考生的高校选择辅助网页，重点呈现高校真实校园生活制度与学生体感。

## 开发原则
1. 先完成最小闭环，再补细节
2. 首版只做前端静态网站
3. 数据结构化，便于筛选与对比
4. 页面必须移动端适配
5. UI 要可信、克制、清晰

## 技术规范
- Next.js App Router
- TypeScript
- Tailwind CSS
- 函数式组件
- 公共类型单独定义
- 不使用 any，除非说明原因

## 开发顺序
1. 初始化项目
2. 建立类型与静态数据
3. 首页学校列表
4. 搜索和筛选
5. 详情页
6. 对比页
